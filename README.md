# Online Meeting System (WebRTC)

## 1. Giới thiệu
Dự án xây dựng hệ thống họp trực tuyến thời gian thực theo mô hình Client–Server tách rời.
Backend đóng vai trò Server Signaling, Frontend là Client chạy trên trình duyệt Web.
Dữ liệu audio/video được truyền trực tiếp giữa các client thông qua WebRTC.

## 2. Công nghệ sử dụng
- Backend: Node.js, Express.js, Socket.IO
- Frontend: HTML, CSS, JavaScript
- WebRTC API
- Canvas API
